# python-eink-weather

### Weather display on an e-ink display with Raspberry Pi
- Waveshare 2.13 inch e-ink display
- Raspberry Pi 3 B
- Python 3
- OpenWeatherMap API

### To run
- install dependencies
- python3 weather.py
